	- 【查看文件树结构】
		安装tree: sudo apt-get install tree
		使用方法及参数：tree -d -L 1
			注：
				tree：显示目录树；
				-d：只显示目录；
				-L：选择显示的目录深度；
				1：只显示一层深度，即不递归子目录；
